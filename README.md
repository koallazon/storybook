# uibook
UI storybook

설치방법
========

### 1. node.js 설치

>[nodejs]: https://nodejs.org/ko/ "Node.js download"

    안정적, 신뢰도 높은 버전 설치


### 2. Git 설치
>[GIT]: https://git-scm.com/ "GIT download"


### 3. 터미널 창

##### Git 사용자 정보 설정
```git config --global user.name``` "사용자명"

```git config --global user.email``` "메일주소"


##### 소스 다운로드
```git clone https://github.com/koallazon/uibook.git```


##### 설치
```npm install```


##### 실행
```npm dev```


##### 배포
```npm build-storybook```
